# Алгоритмы

## Реализованные

### Tower placing
Строим 'башенку' из всех предметов в точке (0, 0).

### Random placing
Для каждого нового предмета его координаты (x, y) выбираются случайно и равномерно, а z-координата определяется по (x, y) - самая верхняя из горизонтальных граней, имеющих непустое пересечение с новым предметом в проекции на (x, y).

### Greedy random placing
Для каждого нового предмета:
- Бросаем 1000 точек (x, y)
- Выбираем ту, для которой получаем наименьший z

### Lowest-fit
Для каждого нового предмета:
- Перебираем всевозможные (x, y) такие, что новый предмет в таких координатах мог бы касаться двух других в проекции на (X, Y)
- Выбираем позицию с наименьшим z

### Min coverage
Для предмета (dx, dy, dz)
- Перебираем всевозможные угловые позиции (x, y)
- Для позиции (x, y, z) посчитаем какой объём она закрывает:
    1. Считая суммарный объём пустот в призме (x, y, 0, x + dx, y + dy, z)
    1. Считая объём "свободных" пустот (таких, что над ними нет предметов)

## Идеи

**Без перемещений**
Жадные:
- Каждый предмет кладём чтобы минимизировать некоторую функцию
    - Минимизируем z-координату нового предмета (или, что то же самое - среднюю z-координату всех предметов)
        - Перебираем угловые позиции
    - Минимизируем "закрываемое" новым предметом пространство
        - Перебираем угловые позиции (перебирая пары угловых предметов)
        - Метрика:
            - Объём (не занятый предметами), блокируемый новым предметом
            - Объём над "картой высот" и под новым предметом (т.е. объём, который мог быть действительно занят)
    - Минимизируем "кривизну" результирующей поверхности
    - Минимизируем среднюю высоту результирующей поверхности

**С перемещениями**
Жадные:
- Предметы кладём **и перемещаем** чтобы минимизировать некоторую функцию
    - Cреднюю z-координату всех предметов
    - Минимизируем "закрываемое" предметами пространство
    - Минимизируем "кривизну" результирующей поверхности
    - Минимизируем среднюю высоту результирующей поверхности

Определяем проблемные "зоны" (с наименьшим заполнением или аналогичной метрикой) и для нескольких проблемных зон и новых предметов пробуем переставить предметы так, чтобы получить хорошее решение (т.е. исправляем проблемные зоны).
- Выделяем проблемные зоны / предметы
- Подбираем лучшее размещение этих предметов
    - С т.з. некоторой метрики
    - С помощью:
        - Перебора с отсечениями
        - Генетического алгоритма
        - Отжига
        - Берём все проблемные предметы, достаём какие можем, для случайных перестановок этих предметов пробуем их переразместить, если метрика улучшилась - запускаем переразмещение
- Производим перестановку
